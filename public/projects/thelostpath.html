The Lost Path

Play on itch.io

This game was created in one week at the occasion of 3rd Nokia 3310 game jam.

The game had to respect the following constrains:

  • - Screen size: 84x48 pixels
  • - Using only 2 colors (the one that corresponds to Nokia's screen)
  • - Monophonic sounds

Technologies Used

  • JavaScript
