The Lost Path

Play on itch.io

This game was created in one week at the occasion of GB game jam 8. GameBoy game created during GB Jam 8.

The game had to respect the following constrains:

  • - Display only 4 colors at screen at the same time
  • - Respect GameBoy aspect-ratio : 160x144 px
  • - Respect GameBoy controls

Technologies Used
