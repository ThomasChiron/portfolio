Verbatim extraction & correction interface

During my internship at LS2N lab, I had the opportunity to create from scratch both the backend and frontend of an application for meeting automatic transcription and correction.

The meeting can be provided either as audio or video and its audio content is fully transcripted thanks to Whisper model. All the speakers of the audio are detected and separated to fit the closest to a human audio transcription with multiple speakers. The program can recognize the speakers accross audio extracts and keeps speakers data in a database.

The web app allows to visualize the previously processed results (transcription) and to listen back to the meeting at the same time of reading the transcription so the user can correct easily the verbatim. The web app features multi-user (concurrent) transcription visualization and correction so multiple people can edit the same time.

Technologies Used

  • Javascript
  • C++
  • OpenCV
