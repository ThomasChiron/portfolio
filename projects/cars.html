Self driving cars

This project was the occasion to explore genetic algorithm with neural networks for driving cars through a circuit. All the calculations are done on browser thanks to tensorflowJS WASM backend.

Each car of the first population is initialized with a random neural network that takes driving decisions according to the car field of view. When either the simulation time exceeds a limit or the whole population is crashed, a new population is created. This new population is created from the previous one by breeding the best individuals and randomly mutating them.

This project could evolve later as a game.

Technologies Used

  • JavaScript
  • Typescript
  • TensorflowJS
