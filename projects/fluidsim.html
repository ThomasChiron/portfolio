Fluid simulation

This is an incompressible fluid simulation based on Navier Stokes equations. The project was first made in JavaScript for real time browser rendering. It was then remade in C++ with OpenCV in order to reach better simulation frame rate.

Technologies Used

  • C++
  • OpenCV
